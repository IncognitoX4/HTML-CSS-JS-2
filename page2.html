Variables

Variables are names that store values.

Variables are case-sensitive.

Variables are dynamic.

Variables can be declared and assigned.

Data Types

Data types are categories that define the type of data that a variable can store.

Data types are case-sensitive.

Data types are dynamic.

Control flow is the order in which statements are executed.

Control flow is conditional.

Control flow is iterative.

Strings

Strings are sequences of characters.

Strings are enclosed in single or double quotes.

Integers

Integers are whole numbers.

Floats

Floats are decimal numbers.

Booleans

Booleans are True or False.

Lists

Lists are ordered and changeable.

Tuples

Tuples are ordered and unchangeable.

Dictionaries

Dictionaries are unordered and changeable.

Sets

Sets are unordered and unchangeable.

Ranges

Ranges are ordered and unchangeable.

Functions

Functions are reusable blocks of code that can be called multiple times.

Loops

Loops are repetitive blocks of code that can be called multiple times.

Conditionals

Conditionals are decision-making blocks of code that can be called multiple times.

Modules

Modules are collections of functions and constants that can be import and exported.

Classes

Classes are blueprints for creating objects.

Exceptions

Exceptions are errors that occur during program execution.

Regular Expressions

Regular Expressions are patterns for matching text.
